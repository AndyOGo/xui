XUI App Example

Welcome to XUI. A set of defaults for simplified mobile web app guis. Include xui.app.js and the following behaviors are enabled:

  • lists with links become link groups as in the native guis
  • relative links loaded via ajax and animated in (with a left to right swipe by default)
  • animations can be changed with class="left-to-right|right-to-left|top-to-bottom|bottom-to-top"
  • complete paths will open in mobile safari
  • skinable via css
