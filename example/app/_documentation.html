Certaldo

A micro MVC framework based on XUI.js for device neutral mobile web applications. Web standards friendly. Everything you need. Nothing you do not.

Controllers

  • can navigate to either a #anchor or a relative external url of content via an xhr call
  • automatically maps an href to a regex/function

Models

Key/Value Store

  • object oriented
  • one database with many key value store tables
  • used internally for offline sync
  • based on silentrob's excellent sql lib

Offline Syncronisation

  • application logic is stored in the database
  • application markup is stored in the database
  • html 5 pragma meta cache
  • checks to see if the app is online, if so, it checks for a newer version

Views

Templating

  • templates begining with an underscore are considered html fragments (eg. _person.html )
  • JSONPath is used to map JSON objects to DOM nodes

Native Theming

  • default themes automatically detects the devices for native guis: iPhone, Android, Palm Pre native
  • dead simple to implment your own
  • sprited for max speed

Other nifty features

  • automatically hides navitagation elements such as the url bar
  • iphone springboard friendly meta tags
  • automatic handling of device orientation events
